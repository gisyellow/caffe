A modern web map application. Built with [Angular](https://angular.io/) and 
[ESRI](https://developers.arcgis.com/).  

## Getting Started
Clone the repository
```
  git clone https://github.com/zkristov/caffe.git
```
Change directories
```
  cd caffe
```
Install dev dependencies
```
  npm install
```
Run test suite
```
  npm run test:once
```
